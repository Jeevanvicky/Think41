# Sri Krishna Hardware Website

A modern, professional website for Sri Krishna Hardware - a premium hardware and agricultural solutions provider.

## 🌟 Features

- **Modern Design**: Clean, professional interface with glass morphism effects
- **Responsive Layout**: Fully responsive design that works on all devices
- **Animated Background**: Dynamic gradient background with smooth animations
- **Three Portal System**: Separate login portals for Admin, Customer, and Retailer access
- **Product Showcase**: Interactive product cards with hover effects
- **Performance Optimized**: Lightweight code with efficient animations

## 📁 Project Structure

```
├── index.html              # Main homepage
├── styles/
│   └── main.css            # Main stylesheet
├── scripts/
│   └── main.js             # JavaScript functionality
├── login/
│   ├── admin-login.html    # Admin portal login
│   ├── customer-login.html # Customer portal login
│   └── retailer-login.html # Retailer portal login
└── README.md               # Project documentation
```

## 🎨 Customization

### Colors
Update the CSS custom properties in `styles/main.css`:
```css
:root {
    --primary-color: #ff006e;
    --secondary-color: #8338ec;
    --accent-color: #06ffa5;
}
```

### Content
- Update company information in `index.html`
- Modify product details in the products section
- Update contact information in the footer

### Styling
- Adjust animations in `styles/main.css`
- Modify card layouts and effects
- Update responsive breakpoints
